CanvasStack Rendering Pipeline

6 stacked <canvas> elements composited via CSS (bottom to top) — each rendered independently via useLayerEffect

Layer Stack (top → bottom)
6. Selection
Dashed outlines, resize handles, labels for selected objects
always visible
5. Plantings
Plant icons (PNG/fallback circle), spacing borders, footprint circles, labels, overlays
togglable + opacity
4. Zones
Colored rectangles with dashed border, pattern overlays, labels
togglable + opacity
3. Structures
Raised beds, pots, planters, surfaces — with wall fills and patterns
togglable + opacity
2. Blueprint
User-uploaded reference image positioned in world space
togglable + opacity
1. Grid
Infinite grid lines + garden plot border (mix-blend-mode: multiply)
always visible
0. Ground (CSS background)
Solid color from garden.groundColor applied to container div

Structure Layer Steps

  1. Clear canvas
  2. Apply layer globalAlpha (opacity)
  3. Filter out hidden IDs (drag overlay)
  4. For each structure: world→screen transform
  5. Draw fill (rect or ellipse based on shape/type)
  6. Draw inner wall fill (raised-bed/pot types)
  7. Render pattern overlay (chunks, hatch)
  8. Draw highlight ring if highlight > 0
  9. Render label below object
  10. Draw overlay structures (drag preview, reduced alpha)

Planting Layer Steps

  1. Clear canvas
  2. Apply layer globalAlpha
  3. Build parent lookup (zones + container structures)
  4. Group plantings by parent container
  5. Per container group: clip to inner wall bounds
  6. Draw spacing border (dashed rect/circle)
  7. Draw highlight ring from overlay config
  8. Render plant icon (renderPlant: PNG image or fallback circle)
  9. Draw measurement labels if enabled
  10. Draw hover highlight ring if highlight > 0
  11. Restore clip, collect label candidates
  12. Second pass: render non-overlapping labels (occlusion test)
  13. Redraw container inner borders on top
  14. Draw overlay plantings (drag preview)

Zone Layer Steps

  1. Clear canvas
  2. Apply layer globalAlpha
  3. Sort zones by zIndex
  4. For each zone: fill rect with zone.color
  5. Draw dashed green border
  6. Render pattern overlay if configured
  7. Draw highlight ring if highlight > 0
  8. Render label below zone
  9. Draw overlay zones (drag preview)

Selection Layer Steps

  1. Clear canvas
  2. Render selected plantings as dashed circles
  3. Render selected structures/zones with blue dashed outline
  4. Draw 8 resize handles on bounding box
  5. Render label below selected object
Key Architecture Concepts
LayerRenderer base class manages highlight animation (flash/hover)
useLayerEffect hook handles DPR scaling + visibility gating
Drag overlays render ghost objects with reduced opacity per layer
Plant icons are async-loaded PNGs; iconTick forces re-render on load

Orchestration: CanvasStack.tsx

CanvasStack owns 6 <canvas> refs stacked absolutely. Each is driven by a useLayerEffect call that: (1) sizes the canvas to width * dpr, (2) scales the context by dpr, (3) calls the render function only if visible. Layers 3-5 use persistent LayerRenderer instances that hold animation state (flash/hover highlight). The renderers call onInvalidate to trigger a React re-render when animating. State flows from Zustand stores (useGardenStore, useUiStore) → renderer properties → useLayerEffect deps → redraw.
